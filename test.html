Yêu cầu tạo 5 API Endpoint cho language dưới đây

  • Đọc tất cả ngôn ngữ hiện có (không cần làm phân trang, vì số lượng language không quá nhiều): GET /languages

  • Đọc chi tiết ngôn ngữ: GET /languages/:languageId

  • Tạo ngôn ngữ mới: POST /languages

  • Sửa ngôn ngữ: PUT /languages

  • Xóa ngôn ngữ: DELETE /languages/:languageId

Vì chúng ta chưa làm chức năng Permission nên là tại bước này, các bạn cứ cho phép login vào là dùng được các API trên. Khi nào làm Permission thì chúng ta sẽ quay lại cập nhật
